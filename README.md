# 🔍 Palindrome Checker (Java - Word Only)

A simple Java program that checks whether a given **single word** is a palindrome.  
The program strictly validates input and **rejects phrases, spaces, and special characters**.

---

## 📖 Overview
This project focuses on validating palindromes in their simplest form — single words.  
Unlike advanced implementations, this version ensures input contains only alphabetic characters and does not allow phrases or sentences.

---

## ⚙️ Features
- ✅ Accepts only single-word input  
- ❌ Rejects phrases with spaces  
- ❌ Rejects punctuation and special characters  
- 🔤 Case-insensitive comparison  
- ⚡ Fast and simple logic  

---

## 🛠️ Technologies Used
- Java  
- String Handling  
- Loops & Conditional Statements  

---

## 🚫 Input Rules
- Only alphabets allowed (A–Z, a–z)  
- No spaces or symbols  
- Example of invalid input:
  - `hello world` ❌  
  - `madam!` ❌
